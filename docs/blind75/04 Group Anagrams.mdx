# 04. Group Anagrams

## Soal

Diberikan sebuah array string `strs`, kelompokkan **anagram-anagram** tersebut bersama. Kamu bisa mengembalikan jawaban dalam **urutan apa pun**.

**Anagram** adalah sebuah kata atau frasa yang dibentuk dengan menyusun ulang huruf-huruf dari kata atau frasa lain, biasanya menggunakan semua huruf asli tepat satu kali.

---

## Contoh

### Contoh 1

**Input:** `strs = ["eat", "tea", "tan", "ate", "nat", "bat"]`  
**Output:** `[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]`

### Contoh 2

**Input:** `strs = [""]`  
**Output:** `[[""]]`

### Contoh 3

**Input:** `strs = ["a"]`  
**Output:** `[["a"]]`

---

## Batasan

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` terdiri dari huruf kecil bahasa Inggris.

---

## Solusi

### Solusi 1: Sorting + Hash Table

**Pendekatan:**
1. Telusuri array string, urutkan setiap string berdasarkan urutan kamus karakter untuk mendapatkan string baru
2. Gunakan string yang sudah diurutkan sebagai `key` dan `[str]` sebagai `value`, simpan dalam hash table
3. Ketika menemukan `key` yang sama, tambahkan string asli ke `value` yang bersesuaian

**Penelusuran Contoh:**

Untuk `strs = ["eat", "tea", "tan", "ate", "nat", "bat"]`:

| Key     | Value                     |
|---------|---------------------------|
| `"aet"` | `["eat", "tea", "ate"]`   |
| `"ant"` | `["tan", "nat"]`          |
| `"abt"` | `["bat"]`                 |

**Kompleksitas Waktu:** O(n x k log k), di mana n = panjang array, k = panjang string maksimum  
**Kompleksitas Ruang:** O(n x k)

```javascript
class Solution {
    /**
     * @param {string[]} strs
     * @return {string[][]}
     */
    groupAnagrams(strs) {
        const map = new Map();

        for (const str of strs) {
            const sortedStr = str.split('').sort().join('');

            if (!map.has(sortedStr)) {
                map.set(sortedStr, []);
            }

            map.get(sortedStr).push(str);
        }

        return Array.from(map.values());
    }
}
```

### Solusi 2: Penghitungan Karakter

**Pendekatan:**
Alih-alih sorting, hitung frekuensi karakter dan gunakan hitungan tersebut sebagai key.

**Kompleksitas Waktu:** O(n x k), di mana n = panjang array, k = panjang string maksimum  
**Kompleksitas Ruang:** O(n x k)

```javascript
class Solution {
    groupAnagrams(strs) {
        const map = new Map();

        for (const str of strs) {
            const count = Array(26).fill(0);
            
            for (const char of str) {
                count[char.charCodeAt(0) - 'a'.charCodeAt(0)]++;
            }
            
            const key = count.join('#');
            
            if (!map.has(key)) {
                map.set(key, []);
            }
            
            map.get(key).push(str);
        }

        return Array.from(map.values());
    }
}
```
