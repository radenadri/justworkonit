# 05. Top K Frequent Elements

## Soal

**Kesulitan:** Medium

Diberikan sebuah array integer yang tidak kosong, kembalikan **k** elemen yang paling sering muncul.

---

## Contoh

### Contoh 1

**Input:** `nums = [1, 1, 1, 2, 2, 3]`, `k = 2`  
**Output:** `[1, 2]`

### Contoh 2

**Input:** `nums = [1]`, `k = 1`  
**Output:** `[1]`

---

## Batasan

- Kamu boleh berasumsi `k` selalu valid: `1 <= k <= jumlah elemen unik`
- Kompleksitas waktu algoritmamu **harus** lebih baik dari O(n log n), di mana n adalah ukuran array

---

## Solusi

### Pendekatan: Hash Map + Sorting

**Strategi:**
1. **Hitung frekuensi:** Gunakan hash map untuk menghitung kemunculan setiap angka
2. **Urutkan berdasarkan frekuensi:** Konversi ke array dan urutkan berdasarkan hitungan secara menurun
3. **Pilih top k:** Ambil k elemen pertama

> **Soal Terkait:** [692. Top K Frequent Words](https://leetcode.com/problems/top-k-frequent-words/)

**Kompleksitas Waktu:** O(n log n) - karena pengurutan  
**Kompleksitas Ruang:** O(n)

```javascript
class Solution {
    /**
     * @param {number[]} nums
     * @param {number} k
     * @return {number[]}
     */
    topKFrequent(nums, k) {
        const hashMap = new Map();

        for (const num of nums) {
            hashMap.set(num, (hashMap.get(num) || 0) + 1);
        }

        const list = [...hashMap];
        list.sort((a, b) => b[1] - a[1]);

        const ans = [];
        for (let i = 0; i < k; i++) {
            ans.push(list[i][0]);
        }

        return ans;
    }
}
```

### Alternatif: Bucket Sort (Waktu O(n))

Untuk kompleksitas waktu yang lebih baik, gunakan bucket sort di mana indeks merepresentasikan frekuensi:

```javascript
class Solution {
    topKFrequent(nums, k) {
        const freqMap = new Map();
        
        for (const num of nums) {
            freqMap.set(num, (freqMap.get(num) || 0) + 1);
        }

        // Buat bucket di mana indeks = frekuensi
        const buckets = Array(nums.length + 1).fill(null).map(() => []);
        
        for (const [num, freq] of freqMap) {
            buckets[freq].push(num);
        }

        // Kumpulkan k elemen paling sering dari bucket (tertinggi ke terendah)
        const result = [];
        for (let i = buckets.length - 1; i >= 0 && result.length < k; i--) {
            result.push(...buckets[i]);
        }

        return result.slice(0, k);
    }
}
```

**Kompleksitas Waktu:** O(n)  
**Kompleksitas Ruang:** O(n)

---

## Kasus Tepi

- **Frekuensi seri:** Ketika beberapa nilai memiliki hitungan yang sama, urutan apa pun diterima
- **k sama dengan jumlah elemen unik:** Kembalikan semua elemen unik diurutkan berdasarkan frekuensi
