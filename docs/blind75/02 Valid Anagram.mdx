# 02. Valid Anagram

## Soal

Diberikan dua string `s` dan `t`, kembalikan `true` jika `t` adalah anagram dari `s`, dan `false` jika bukan.

**Anagram** adalah sebuah kata atau frasa yang dibentuk dengan menyusun ulang huruf-huruf dari kata atau frasa lain, menggunakan semua huruf asli tepat satu kali.

---

## Contoh

### Contoh 1

**Input:** `s = "anagram"`, `t = "nagaram"`  
**Output:** `true`

### Contoh 2

**Input:** `s = "rat"`, `t = "car"`  
**Output:** `false`

---

## Batasan

- `1 <= s.length, t.length <= 5 * 10^4`
- `s` dan `t` terdiri dari huruf kecil bahasa Inggris.

---

## Tindak Lanjut

Bagaimana jika input mengandung karakter Unicode? Bagaimana kamu menyesuaikan solusimu?

> **Jawaban:** Gunakan `Map` sebagai pengganti array berukuran tetap untuk menghitung frekuensi karakter, karena Unicode memiliki himpunan karakter yang jauh lebih besar.

---

## Solusi

### Pendekatan: Penghitungan Karakter

1. Jika panjang string berbeda, langsung kembalikan `false`
2. Gunakan array berisi 26 elemen untuk menghitung frekuensi karakter
3. Tambahkan hitungan untuk karakter di `s`, kurangi untuk karakter di `t`
4. Jika semua hitungan nol, maka kedua string adalah anagram

**Kompleksitas Waktu:** O(n)  
**Kompleksitas Ruang:** O(1) - array tetap berisi 26 elemen

```javascript
class Solution {
    /**
     * @param {string} s
     * @param {string} t
     * @return {boolean}
     */
    isAnagram(s, t) {
        if (s.length !== t.length) {
            return false;
        }

        const cnt = Array(26).fill(0);

        for (let i = 0; i < s.length; ++i) {
            ++cnt[s.charCodeAt(i) - 'a'.charCodeAt(0)];
            --cnt[t.charCodeAt(i) - 'a'.charCodeAt(0)];
        }

        return cnt.every(x => x === 0);
    }
}
```

### Alternatif: Sorting

```javascript
class Solution {
    isAnagram(s, t) {
        if (s.length !== t.length) return false;
        return s.split('').sort().join('') === t.split('').sort().join('');
    }
}
```

**Kompleksitas Waktu:** O(n log n)  
**Kompleksitas Ruang:** O(n)
