# 03. Two Sum

## Soal

Diberikan sebuah array integer `nums` dan sebuah integer `target`, kembalikan *indeks dari dua angka yang jika dijumlahkan menghasilkan `target`*.

Kamu boleh berasumsi bahwa setiap input memiliki **tepat satu solusi**, dan kamu tidak boleh menggunakan elemen yang *sama* dua kali.

Kamu bisa mengembalikan jawaban dalam urutan apa pun.

---

## Contoh

### Contoh 1

**Input:** `nums = [2, 7, 11, 15]`, `target = 9`  
**Output:** `[0, 1]`  
**Penjelasan:** Karena `nums[0] + nums[1] == 9`, kita kembalikan `[0, 1]`.

### Contoh 2

**Input:** `nums = [3, 2, 4]`, `target = 6`  
**Output:** `[1, 2]`

### Contoh 3

**Input:** `nums = [3, 3]`, `target = 6`  
**Output:** `[0, 1]`

---

## Batasan

- `2 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
- **Hanya ada satu jawaban yang valid.**

---

## Tindak Lanjut

Bisakah kamu membuat algoritma dengan kompleksitas waktu kurang dari O(n²)?

---

## Solusi

### Pendekatan: Hash Map

1. Iterasi melalui array
2. Untuk setiap elemen `x`, hitung komplemen `y = target - x`
3. Periksa apakah komplemen tersebut ada di hash map
4. Jika ditemukan, kembalikan kedua indeks; jika tidak, simpan elemen saat ini beserta indeksnya

**Kompleksitas Waktu:** O(n)  
**Kompleksitas Ruang:** O(n)

```javascript
class Solution {
    /**
     * @param {number[]} nums
     * @param {number} target
     * @return {number[]}
     */
    twoSum(nums, target) {
        const map = new Map();

        for (let i = 0; i < nums.length; ++i) {
            const x = nums[i];
            const y = target - x;

            if (map.has(y)) {
                return [map.get(y), i];
            }

            map.set(x, i);
        }
    }
}
```

### Brute Force (Tidak Direkomendasikan)

```javascript
class Solution {
    twoSum(nums, target) {
        for (let i = 0; i < nums.length; i++) {
            for (let j = i + 1; j < nums.length; j++) {
                if (nums[i] + nums[j] === target) {
                    return [i, j];
                }
            }
        }
    }
}
```

**Kompleksitas Waktu:** O(n²)  
**Kompleksitas Ruang:** O(1)
