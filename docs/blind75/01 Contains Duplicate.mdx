# 01. Contains Duplicate

## Soal

Diberikan sebuah array integer `nums`, kembalikan `true` jika ada nilai yang muncul **setidaknya dua kali** dalam array, dan kembalikan `false` jika setiap elemen berbeda (unik).

---

## Contoh

### Contoh 1

**Input:** `nums = [1, 2, 3, 1]`  
**Output:** `true`  
**Penjelasan:** Elemen 1 muncul di indeks 0 dan 3.

### Contoh 2

**Input:** `nums = [1, 2, 3, 4]`  
**Output:** `false`  
**Penjelasan:** Semua elemen berbeda.

### Contoh 3

**Input:** `nums = [1, 1, 1, 3, 3, 4, 3, 2, 4, 2]`  
**Output:** `true`

---

## Batasan

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

---

## Solusi

### Pendekatan: Sorting

1. Urutkan array
2. Iterasi dan periksa apakah ada elemen bersebelahan yang sama
3. Jika ditemukan, kembalikan `true`; jika tidak, kembalikan `false`

**Kompleksitas Waktu:** O(n log n) - karena pengurutan  
**Kompleksitas Ruang:** O(1) - pengurutan in-place (atau O(n) tergantung implementasi sort)

```javascript
class Solution {
    /**
     * @param {number[]} nums
     * @return {boolean}
     */
    hasDuplicate(nums) {
        const sortedNums = nums.sort((a, b) => a - b);

        for (let i = 1; i < sortedNums.length; i++) {
            if (sortedNums[i] === sortedNums[i - 1]) {
                return true;
            }
        }

        return false;
    }
}
```

### Alternatif: Hash Set

Pendekatan yang lebih optimal menggunakan Set dengan kompleksitas waktu O(n):

```javascript
class Solution {
    hasDuplicate(nums) {
        const seen = new Set();
        
        for (const num of nums) {
            if (seen.has(num)) {
                return true;
            }
            seen.add(num);
        }
        
        return false;
    }
}
```

**Kompleksitas Waktu:** O(n)  
**Kompleksitas Ruang:** O(n)
