Aturan praktis yang perlu kamu ulang terus:

- **Mulai dari input-output.** Tulis: apa masukannya, apa keluarannya, formatnya, dan contoh kecil.

- **Uji dengan 3 jenis contoh.** Kasus normal, kasus kecil/ekstrem, dan kasus yang "menjebak".

## Contoh Cepat
```js
// Contoh pola berpikir yang dipakai di seluruh seri.
// Masalah: cek apakah semua karakter dalam string unik.
// Input: string s
// Output: boolean
// Edge case: string kosong => true

function semuaKarakterUnik(s) {
  if (typeof s !== "string") return false;

  const seen = new Set();
  for (const ch of s) {
    if (seen.has(ch)) return false;
    seen.add(ch);
  }
  return true;
}

console.log(semuaKarakterUnik("abcd")); // true
console.log(semuaKarakterUnik("abca")); // false
console.log(semuaKarakterUnik("")); // true

// Kompleksitas:
// - Waktu: O(n) karena tiap karakter dicek sekali
// - Memori: O(n) untuk menyimpan karakter yang sudah terlihat
```

## Studi Kasus (Sehari-hari)

Kamu punya daftar username yang baru didaftarkan hari ini. Kamu ingin menolak pendaftaran jika username sudah pernah dipakai sebelumnya.

Terjemahan ke DSA:

- **Input**: array string `usernamesBaru` + set/daftar `usernamesTerdaftar`.
- **Output**: daftar username yang valid / atau boolean "boleh daftar?".
- **Strategi**: gunakan **`Set`** untuk cek/pencarian cepat (cek duplikat) daripada scan array berulang.
- **Edge case**: kapitalisasi ("Ari" vs "ari"), spasi, string kosong, duplikat dalam batch yang sama.

Intinya: begitu kamu butuh "pernah lihat atau belum?" atau "berapa kali muncul?", pikirkan **Set/Map** dulu.

## Latihan
1. Buat fungsi `pairSum(nums, target)` yang mengembalikan `true/false` apakah ada dua angka yang jumlahnya `target`.

   <details>
     <summary>Petunjuk</summary>

     - Saat membaca angka `x`, cek apakah `target - x` sudah pernah terlihat.
     - Gunakan `Set` atau `Map` agar ceknya O(1) rata-rata.
   </details>

2. Buat fungsi `isBalanced(s)` untuk mengecek apakah tanda kurung `()[]{}` seimbang.

   <details>
     <summary>Petunjuk</summary>

     - Gunakan stack; push saat ketemu buka, pop saat ketemu tutup.
     - Buat mapping pasangan tutup -> buka.
   </details>

3. Buat fungsi `binarySearch(arr, x)` untuk array angka yang sudah terurut naik. Kembalikan index atau `-1`.

   <details>
     <summary>Petunjuk</summary>

     - Pakai dua pointer `lo` dan `hi`, lalu hitung `mid`.
     - Hati-hati update batas agar loop tidak infinite.
   </details>

4. Buat fungsi `freqMap(words)` yang menerima array string dan mengembalikan `Map` (atau object) frekuensi kemunculan.

   <details>
     <summary>Petunjuk</summary>

     - Jika kata belum ada di map, mulai dari 1.
     - Kalau sudah ada, tambah 1.
   </details>
