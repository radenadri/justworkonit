# Instalasi & Setup Environment

## Menginstal Rust dengan Rustup

**Rustup** adalah official toolchain manager untuk Rust. Ini menginstal compiler (`rustc`), package manager (`cargo`), dan tools lainnya.

### macOS / Linux

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Setelah instalasi, restart terminal atau jalankan:

```bash
source $HOME/.cargo/env
```

### Windows

Download dan jalankan installer dari [rustup.rs](https://rustup.rs/). Kamu juga membutuhkan **Visual Studio C++ Build Tools**.

### Verifikasi Instalasi

```bash
rustc --version
# rustc 1.XX.0 (hash 2024-XX-XX)

cargo --version
# cargo 1.XX.0 (hash 2024-XX-XX)

rustup --version
# rustup 1.XX.0 (hash 2024-XX-XX)
```

## Komponen Rust Toolchain

| Komponen | Fungsi |
|----------|--------|
| `rustc` | Rust compiler — mengubah `.rs` menjadi binary |
| `cargo` | Package manager dan build system |
| `rustup` | Toolchain version manager |
| `rustfmt` | Code formatter (auto-format) |
| `clippy` | Linter — saran untuk kode yang lebih idiomatik |
| `rust-analyzer` | Language Server Protocol (LSP) untuk IDE |

## Perintah Rustup yang Penting

```bash
# Update Rust ke versi terbaru
rustup update

# Lihat toolchain yang terinstal
rustup show

# Install komponen tambahan
rustup component add rustfmt
rustup component add clippy

# Switch ke nightly (opsional)
rustup install nightly
rustup default nightly

# Kembali ke stable
rustup default stable
```

## Setup IDE — VS Code

### 1. Install Extensions

Buka VS Code, lalu install extensions berikut:

- [ ] **rust-analyzer** — LSP utama untuk Rust (autocomplete, error checking, refactoring)
- [ ] **Even Better TOML** — Syntax highlighting untuk `Cargo.toml`
- [ ] **CodeLLDB** — Debugger untuk Rust

### 2. Konfigurasi settings.json

```json
{
  "rust-analyzer.check.command": "clippy",
  "rust-analyzer.inlayHints.typeHints.enable": true,
  "rust-analyzer.inlayHints.parameterHints.enable": true,
  "[rust]": {
    "editor.formatOnSave": true,
    "editor.defaultFormatter": "rust-lang.rust-analyzer"
  }
}
```

### 3. Fitur rust-analyzer

| Fitur | Shortcut |
|-------|----------|
| Go to definition | `F12` |
| Find references | `Shift+F12` |
| Rename symbol | `F2` |
| Format document | `Shift+Alt+F` |
| Run test at cursor | Click "Run Test" lens |
| Expand macro | `Ctrl+Shift+P` → "Expand Macro" |

## Rustfmt — Code Formatter

Rustfmt memformat kode Rust secara otomatis sesuai style guide resmi.

```bash
# Format satu file
rustfmt src/main.rs

# Format seluruh project via cargo
cargo fmt

# Cek tanpa mengubah file
cargo fmt --check
```

Konfigurasi opsional di `rustfmt.toml`:

```toml
max_width = 100
tab_spaces = 4
edition = "2021"
```

## Clippy — Linter

Clippy memberikan saran untuk menulis kode Rust yang lebih idiomatik dan efisien.

```bash
# Jalankan clippy
cargo clippy

# Clippy dengan semua warnings sebagai error
cargo clippy -- -D warnings

# Fix otomatis (jika memungkinkan)
cargo clippy --fix
```

Contoh saran Clippy:

```rust
// Clippy akan menyarankan:
// ❌ Kurang idiomatik
if x == true {
    // ...
}

// ✅ Lebih idiomatik
if x {
    // ...
}
```

## Troubleshooting

### "command not found: rustc"

```bash
# Pastikan PATH sudah benar
export PATH="$HOME/.cargo/bin:$PATH"

# Tambahkan ke ~/.bashrc atau ~/.zshrc agar permanen
echo 'export PATH="$HOME/.cargo/bin:$PATH"' >> ~/.zshrc
```

### "linker cc not found" (Linux)

```bash
# Ubuntu/Debian
sudo apt install build-essential

# Fedora
sudo dnf install gcc
```

### Update gagal

```bash
# Reset rustup
rustup self update
rustup update
```

### Soal 3 — Clippy Challenge (Sedang)

Buat file `src/main.rs` dengan kode berikut, lalu jalankan `cargo clippy` dan perbaiki semua warnings:

```rust
fn main() {
    let x: i32 = 5;
    let y = x.clone();
    let name = "Rust".to_string();
    if name.len() > 0 {
        println!("Language: {}", name);
    }
    let numbers = vec![1, 2, 3, 4, 5];
    let mut sum = 0;
    for i in 0..numbers.len() {
        sum += numbers[i];
    }
    println!("y = {}, sum = {}", y, sum);
}
```

**Jawaban — Kode setelah diperbaiki:**

```rust
fn main() {
    let x: i32 = 5;
    let y = x; // i32 implements Copy, no need for .clone()
    let name = "Rust".to_string();
    if !name.is_empty() { // Use is_empty() instead of len() > 0
        println!("Language: {name}"); // Use inline format
    }
    let numbers = vec![1, 2, 3, 4, 5];
    let sum: i32 = numbers.iter().sum(); // Use iterator instead of manual indexing
    println!("y = {y}, sum = {sum}");
}
```
