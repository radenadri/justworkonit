# Studi Kasus: REST API dengan Axum

Pada bab terakhir ini, kita akan membangun **REST API lengkap** menggunakan framework **Axum** — salah satu web framework Rust paling populer yang dibangun di atas ekosistem Tokio. Kita akan membuat API untuk manajemen kontak dengan fitur CRUD, validasi, dan error handling yang proper.

## Kenapa Axum?

Axum dikembangkan oleh tim Tokio dan memiliki beberapa keunggulan:

- **Type-safe routing** — extractor pattern yang powerful
- **Middleware via Tower** — ekosistem middleware yang mature
- **Async-native** — built on top of Tokio
- **Ergonomic** — API yang bersih dan idiomatic Rust
- **Performance** — salah satu yang tercepat di benchmark web framework

## Model Data

File `src/models.rs` — definisi struct untuk Contact:

```rust
use chrono::{DateTime, Utc};
use serde::{Deserialize, Serialize};
use uuid::Uuid;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Contact {
    pub id: Uuid,
    pub name: String,
    pub email: String,
    pub phone: Option<String>,
    pub company: Option<String>,
    pub notes: Option<String>,
    pub created_at: DateTime<Utc>,
    pub updated_at: DateTime<Utc>,
}

#[derive(Debug, Deserialize)]
pub struct CreateContact {
    pub name: String,
    pub email: String,
    pub phone: Option<String>,
    pub company: Option<String>,
    pub notes: Option<String>,
}

#[derive(Debug, Deserialize)]
pub struct UpdateContact {
    pub name: Option<String>,
    pub email: Option<String>,
    pub phone: Option<String>,
    pub company: Option<String>,
    pub notes: Option<String>,
}

#[derive(Debug, Serialize)]
pub struct ApiResponse<T: Serialize> {
    pub success: bool,
    pub data: Option<T>,
    pub message: String,
}

#[derive(Debug, Serialize)]
pub struct PaginatedResponse<T: Serialize> {
    pub success: bool,
    pub data: Vec<T>,
    pub total: usize,
    pub page: usize,
    pub per_page: usize,
}

#[derive(Debug, Deserialize)]
pub struct ListQuery {
    pub page: Option<usize>,
    pub per_page: Option<usize>,
    pub search: Option<String>,
}

impl Contact {
    pub fn new(input: CreateContact) -> Self {
        let now = Utc::now();
        Self {
            id: Uuid::new_v4(),
            name: input.name,
            email: input.email,
            phone: input.phone,
            company: input.company,
            notes: input.notes,
            created_at: now,
            updated_at: now,
        }
    }

    pub fn apply_update(&mut self, update: UpdateContact) {
        if let Some(name) = update.name {
            self.name = name;
        }
        if let Some(email) = update.email {
            self.email = email;
        }
        if update.phone.is_some() {
            self.phone = update.phone;
        }
        if update.company.is_some() {
            self.company = update.company;
        }
        if update.notes.is_some() {
            self.notes = update.notes;
        }
        self.updated_at = Utc::now();
    }
}
```

## Custom Error Types

File `src/errors.rs` — error handling yang proper:

```rust
use axum::{
    http::StatusCode,
    response::{IntoResponse, Response},
    Json,
};
use serde_json::json;

#[derive(Debug)]
pub enum ApiError {
    NotFound(String),
    BadRequest(String),
    Conflict(String),
    Internal(String),
}

impl std::fmt::Display for ApiError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            ApiError::NotFound(msg) => write!(f, "Not Found: {}", msg),
            ApiError::BadRequest(msg) => write!(f, "Bad Request: {}", msg),
            ApiError::Conflict(msg) => write!(f, "Conflict: {}", msg),
            ApiError::Internal(msg) => write!(f, "Internal Error: {}", msg),
        }
    }
}

impl IntoResponse for ApiError {
    fn into_response(self) -> Response {
        let (status, message) = match &self {
            ApiError::NotFound(msg) => (StatusCode::NOT_FOUND, msg.clone()),
            ApiError::BadRequest(msg) => (StatusCode::BAD_REQUEST, msg.clone()),
            ApiError::Conflict(msg) => (StatusCode::CONFLICT, msg.clone()),
            ApiError::Internal(msg) => {
                tracing::error!("Internal error: {}", msg);
                (
                    StatusCode::INTERNAL_SERVER_ERROR,
                    "Internal server error".to_string(),
                )
            }
        };

        let body = json!({
            "success": false,
            "error": message,
            "status": status.as_u16(),
        });

        (status, Json(body)).into_response()
    }
}
```

## In-Memory Store

File `src/store.rs` — thread-safe data store menggunakan Arc dan RwLock:

```rust
use std::collections::HashMap;
use std::sync::Arc;
use tokio::sync::RwLock;
use uuid::Uuid;

use crate::models::{Contact, CreateContact, UpdateContact};
use crate::errors::ApiError;

pub type SharedStore = Arc<ContactStore>;

pub struct ContactStore {
    contacts: RwLock<HashMap<Uuid, Contact>>,
}

impl ContactStore {
    pub fn new() -> Self {
        Self {
            contacts: RwLock::new(HashMap::new()),
        }
    }

    pub async fn create(&self, input: CreateContact) -> Result<Contact, ApiError> {
        // Validasi
        if input.name.trim().is_empty() {
            return Err(ApiError::BadRequest(
                "Name cannot be empty".to_string(),
            ));
        }
        if !input.email.contains('@') {
            return Err(ApiError::BadRequest(
                "Invalid email format".to_string(),
            ));
        }

        // Cek duplikat email
        let contacts = self.contacts.read().await;
        let email_exists = contacts
            .values()
            .any(|c| c.email.to_lowercase() == input.email.to_lowercase());
        drop(contacts);

        if email_exists {
            return Err(ApiError::Conflict(format!(
                "Contact with email '{}' already exists",
                input.email
            )));
        }

        let contact = Contact::new(input);
        let result = contact.clone();

        let mut contacts = self.contacts.write().await;
        contacts.insert(contact.id, contact);

        Ok(result)
    }

    pub async fn get(&self, id: Uuid) -> Result<Contact, ApiError> {
        let contacts = self.contacts.read().await;
        contacts
            .get(&id)
            .cloned()
            .ok_or_else(|| ApiError::NotFound(format!("Contact {} not found", id)))
    }

    pub async fn list(
        &self,
        search: Option<String>,
        page: usize,
        per_page: usize,
    ) -> (Vec<Contact>, usize) {
        let contacts = self.contacts.read().await;

        let mut filtered: Vec<Contact> = if let Some(ref query) = search {
            let q = query.to_lowercase();
            contacts
                .values()
                .filter(|c| {
                    c.name.to_lowercase().contains(&q)
                        || c.email.to_lowercase().contains(&q)
                        || c.company
                            .as_ref()
                            .map(|co| co.to_lowercase().contains(&q))
                            .unwrap_or(false)
                })
                .cloned()
                .collect()
        } else {
            contacts.values().cloned().collect()
        };

        // Sort by created_at descending (newest first)
        filtered.sort_by(|a, b| b.created_at.cmp(&a.created_at));

        let total = filtered.len();
        let start = (page - 1) * per_page;
        let paginated = filtered
            .into_iter()
            .skip(start)
            .take(per_page)
            .collect();

        (paginated, total)
    }

    pub async fn update(
        &self,
        id: Uuid,
        input: UpdateContact,
    ) -> Result<Contact, ApiError> {
        // Validasi email jika di-update
        if let Some(ref email) = input.email {
            if !email.contains('@') {
                return Err(ApiError::BadRequest(
                    "Invalid email format".to_string(),
                ));
            }

            // Cek duplikat email (kecuali contact ini sendiri)
            let contacts = self.contacts.read().await;
            let email_exists = contacts.values().any(|c| {
                c.id != id
                    && c.email.to_lowercase() == email.to_lowercase()
            });
            drop(contacts);

            if email_exists {
                return Err(ApiError::Conflict(format!(
                    "Another contact with email '{}' already exists",
                    email
                )));
            }
        }

        let mut contacts = self.contacts.write().await;
        let contact = contacts
            .get_mut(&id)
            .ok_or_else(|| ApiError::NotFound(format!("Contact {} not found", id)))?;

        contact.apply_update(input);
        Ok(contact.clone())
    }

    pub async fn delete(&self, id: Uuid) -> Result<Contact, ApiError> {
        let mut contacts = self.contacts.write().await;
        contacts
            .remove(&id)
            .ok_or_else(|| ApiError::NotFound(format!("Contact {} not found", id)))
    }

    pub async fn stats(&self) -> HashMap<String, usize> {
        let contacts = self.contacts.read().await;
        let total = contacts.len();
        let with_company = contacts
            .values()
            .filter(|c| c.company.is_some())
            .count();
        let with_phone = contacts
            .values()
            .filter(|c| c.phone.is_some())
            .count();

        let mut stats = HashMap::new();
        stats.insert("total_contacts".to_string(), total);
        stats.insert("with_company".to_string(), with_company);
        stats.insert("with_phone".to_string(), with_phone);
        stats
    }
}
```

```rust
use axum::{
    extract::{Path, Query, State},
    Json,
};
use uuid::Uuid;

use crate::errors::ApiError;
use crate::models::{
    ApiResponse, CreateContact, ListQuery, PaginatedResponse,
    UpdateContact,
};
use crate::store::SharedStore;

// GET /api/contacts
pub async fn list_contacts(
    State(store): State<SharedStore>,
    Query(query): Query<ListQuery>,
) -> Result<Json<PaginatedResponse<serde_json::Value>>, ApiError> {
    let page = query.page.unwrap_or(1).max(1);
    let per_page = query.per_page.unwrap_or(10).min(100);

    let (contacts, total) = store.list(query.search, page, per_page).await;

    let data: Vec<serde_json::Value> = contacts
        .into_iter()
        .map(|c| serde_json::to_value(c).unwrap())
        .collect();

    Ok(Json(PaginatedResponse {
        success: true,
        data,
        total,
        page,
        per_page,
    }))
}

// POST /api/contacts
pub async fn create_contact(
    State(store): State<SharedStore>,
    Json(input): Json<CreateContact>,
) -> Result<(axum::http::StatusCode, Json<ApiResponse<serde_json::Value>>), ApiError> {
    let contact = store.create(input).await?;

    Ok((
        axum::http::StatusCode::CREATED,
        Json(ApiResponse {
            success: true,

}
```

## Main Entry Point

File `src/main.rs` — setup router, middleware, dan server:

```rust
mod errors;
mod handlers;
mod models;
mod store;

use axum::{
    routing::{get, post, put, delete},
    Router,
};
use std::sync::Arc;
use tower_http::cors::{Any, CorsLayer};
use tower_http::trace::TraceLayer;

use store::ContactStore;

#[tokio::main]
async fn main() {
    // Setup tracing
    tracing_subscriber::fmt()
        .with_target(false)
        .compact()
        .init();

    // Setup shared state
    let store = Arc::new(ContactStore::new());

    // Setup CORS
    let cors = CorsLayer::new()
        .allow_origin(Any)
        .allow_methods(Any)
        .allow_headers(Any);

    // Build router
    let app = Router::new()
        .route("/health", get(handlers::health_check))
        .route("/api/stats", get(handlers::get_stats))
        .route(
            "/api/contacts",
            get(handlers::list_contacts).post(handlers::create_contact),
        )
        .route(
            "/api/contacts/{id}",
            get(handlers::get_contact)
                .put(handlers::update_contact)
                .delete(handlers::delete_contact),
        )
        .layer(cors)
        .layer(TraceLayer::new_for_http())
        .with_state(store);

    // Start server
    let addr = "0.0.0.0:3000";
    tracing::info!("Server running on http://{}", addr);

    let listener = tokio::net::TcpListener::bind(addr).await.unwrap();
    axum::serve(listener, app).await.unwrap();
}
```

## Penjelasan Konsep Penting

### Axum Extractors

Extractor adalah cara Axum mengambil data dari request secara type-safe:

```rust
// State extractor — shared application state
State(store): State<SharedStore>

// Path extractor — URL parameters
Path(id): Path<Uuid>

// Query extractor — query string parameters
Query(query): Query<ListQuery>

// Json extractor — request body
Json(input): Json<CreateContact>
```

Setiap extractor secara otomatis melakukan parsing dan validasi. Jika parsing gagal, Axum mengembalikan error response tanpa handler kita perlu menangani secara manual.

### State Management dengan Arc

```rust
// Arc memungkinkan shared ownership across threads
let store = Arc::new(ContactStore::new());

// .with_state() membuat store tersedia di semua handlers
Router::new()
    .route("/api/contacts", get(list_contacts))
    .with_state(store);
```

### RwLock vs Mutex

Kita menggunakan `tokio::sync::RwLock` daripada `Mutex` karena:

- **RwLock** — multiple readers OR single writer. Cocok untuk read-heavy workloads
- **Mutex** — single accessor at a time. Simpler tapi bisa jadi bottleneck

```rust
// Multiple requests bisa baca bersamaan
let contacts = self.contacts.read().await;

// Hanya satu request bisa menulis
let mut contacts = self.contacts.write().await;
```

### Tower Middleware

Axum menggunakan Tower ecosystem untuk middleware:

```rust
// CORS — mengizinkan cross-origin requests
let cors = CorsLayer::new()
    .allow_origin(Any)
    .allow_methods(Any)
    .allow_headers(Any);

// Tracing — logging setiap request
let trace = TraceLayer::new_for_http();

// Pasang middleware ke router
Router::new()
    .layer(cors)
    .layer(trace)
```

### Soal 2 — Implementasi Bulk Delete (Sedang)

Buat endpoint `POST /api/contacts/bulk-delete` yang menerima array UUID dan menghapus semua contacts tersebut.

**Jawaban:**

```rust
// Model untuk bulk delete request
#[derive(Debug, Deserialize)]
pub struct BulkDeleteRequest {
    pub ids: Vec<Uuid>,
}

#[derive(Debug, Serialize)]
pub struct BulkDeleteResponse {
    pub deleted: usize,
    pub not_found: Vec<String>,
}

// Handler
pub async fn bulk_delete(
    State(store): State<SharedStore>,
    Json(input): Json<BulkDeleteRequest>,
) -> Result<Json<ApiResponse<BulkDeleteResponse>>, ApiError> {
    if input.ids.is_empty() {
        return Err(ApiError::BadRequest("No IDs provided".to_string()));
    }

    let mut contacts = store.contacts.write().await;
    let mut deleted = 0;
    let mut not_found = Vec::new();

    for id in input.ids {
        if contacts.remove(&id).is_some() {
            deleted += 1;
        } else {
            not_found.push(id.to_string());
        }
    }

    Ok(Json(ApiResponse {
        success: true,
        data: Some(BulkDeleteResponse { deleted, not_found }),
        message: format!("{} contacts deleted", deleted),
    }))
}
```

```rust
use axum::{

use std::time::Instant;

pub async fn request_logger(
    request: Request,
    next: Next,
) -> Response {
    let method = request.method().clone();
    let uri = request.uri().clone();
    let start = Instant::now();

    let response = next.run(request).await;

    let duration = start.elapsed();
    let status = response.status();

    tracing::info!(

        method,
        uri.path(),
        status.as_u16(),
        duration
```

### Soal 5 — Tambahkan Import dari JSON (Sulit)

Buat endpoint `POST /api/contacts/import` yang menerima array contacts dan meng-import semuanya, skip yang duplikat email.

**Jawaban:**

```rust
#[derive(Debug, Serialize)]
pub struct ImportResult {
    pub imported: usize,
    pub skipped: usize,
    pub errors: Vec<String>,
}

pub async fn import_contacts(
    State(store): State<SharedStore>,
    Json(inputs): Json<Vec<CreateContact>>,
) -> Result<Json<ApiResponse<ImportResult>>, ApiError> {
    if inputs.is_empty() {
        return Err(ApiError::BadRequest("No contacts to import".to_string()));
    }

    let mut imported = 0;
    let mut skipped = 0;
    let mut errors = Vec::new();

    for (i, input) in inputs.into_iter().enumerate() {
        match store.create(input).await {
            Ok(_) => imported += 1,
            Err(ApiError::Conflict(msg)) => {
                skipped += 1;
                errors.push(format!("Row {}: {}", i + 1, msg));
            }
            Err(ApiError::BadRequest(msg)) => {
                skipped += 1;
                errors.push(format!("Row {}: {}", i + 1, msg));
            }
            Err(e) => {
                errors.push(format!("Row {}: {}", i + 1, e));
            }
        }
    }

    Ok(Json(ApiResponse {
        success: true,
        data: Some(ImportResult {
            imported,
            skipped,
            errors,
        }),
        message: format!(
            "{} imported, {} skipped",
            imported, skipped
        ),
    }))
}
```

### Soal 6 — Implementasi Rate Limiting Sederhana (Sulit)

Buat middleware rate limiter sederhana yang membatasi 100 request per menit per IP.

**Jawaban:**

```rust
use std::collections::HashMap;
use std::sync::Arc;
use std::time::{Duration, Instant};
use tokio::sync::Mutex;
use axum::{
    extract::ConnectInfo,
    middleware::Next,
    extract::Request,
    response::{IntoResponse, Response},
    http::StatusCode,
    Json,
};
use std::net::SocketAddr;

#[derive(Clone)]
pub struct RateLimiter {
    requests: Arc<Mutex<HashMap<String, Vec<Instant>>>>,
    max_requests: usize,
    window: Duration,
}

impl RateLimiter {
    pub fn new(max_requests: usize, window_secs: u64) -> Self {
        Self {
            requests: Arc::new(Mutex::new(HashMap::new())),
            max_requests,
            window: Duration::from_secs(window_secs),
        }
    }
}

pub async fn rate_limit(
    ConnectInfo(addr): ConnectInfo<SocketAddr>,
    State(limiter): State<RateLimiter>,
    request: Request,
    next: Next,
) -> Result<Response, Response> {
    let ip = addr.ip().to_string();
    let now = Instant::now();

    let mut requests = limiter.requests.lock().await;
    let entry = requests.entry(ip).or_insert_with(Vec::new);

    // Hapus request yang sudah di luar window
    entry.retain(|t| now.duration_since(*t) < limiter.window);

    if entry.len() >= limiter.max_requests {
        let body = serde_json::json!({
            "success": false,
            "error": "Too many requests. Please try again later.",
        });
        return Err((StatusCode::TOO_MANY_REQUESTS, Json(body)).into_response());
    }

    entry.push(now);
    drop(requests);

    Ok(next.run(request).await)
}
```
