# Hello World & Cargo

## Program Rust Pertama

Buat file `main.rs`:

```rust
fn main() {
    println!("Hello, world!");
}
```

Compile dan jalankan:

```bash
rustc main.rs
./main
# Output: Hello, world!
```

### Anatomi Program Rust

```rust
fn main() {          // Entry point — setiap program Rust dimulai dari fn main()
    println!("Hello, world!"); // println! adalah macro (bukan function — perhatikan tanda !)
}                    // Setiap statement diakhiri semicolon (;)
```

**Catatan penting:**
- `fn` — keyword untuk mendeklarasikan function
- `main()` — function spesial yang menjadi entry point
- `println!` — macro untuk print ke stdout (tanda `!` menandakan ini macro)
- Semicolon (`;`) wajib di akhir statement

## Cargo — Build System & Package Manager

Cargo adalah tool utama untuk mengelola project Rust. Hampir semua project Rust menggunakan Cargo.

### Membuat Project Baru

```bash
# Buat project binary (executable)
cargo new nama_project
cd nama_project

# Buat project library
cargo new nama_library --lib
```

### Struktur Project Cargo

```
nama_project/
├── Cargo.toml       # Manifest — metadata dan dependencies
├── Cargo.lock       # Lock file (auto-generated, jangan edit manual)
├── src/
│   └── main.rs      # Entry point (binary) atau lib.rs (library)
├── target/          # Build output (auto-generated)
└── .gitignore       # Git ignore (auto-generated)
```

### Cargo.toml — Project Manifest

```toml
[package]
name = "nama_project"
version = "0.1.0"
edition = "2021"          # Rust edition (2015, 2018, 2021)
authors = ["Nama Kamu <email@example.com>"]
description = "Deskripsi project"

[dependencies]
# Dependencies ditambahkan di sini
# serde = "1.0"
# tokio = { version = "1", features = ["full"] }

[dev-dependencies]
# Dependencies untuk testing
# criterion = "0.5"
```

### Perintah Cargo yang Penting

| Perintah | Fungsi |
|----------|--------|
| `cargo new` | Buat project baru |
| `cargo build` | Compile project (debug mode) |
| `cargo build --release` | Compile dengan optimasi (release mode) |
| `cargo run` | Compile dan jalankan |
| `cargo check` | Cek error tanpa compile (lebih cepat) |
| `cargo test` | Jalankan semua test |
| `cargo doc --open` | Generate dan buka dokumentasi |
| `cargo fmt` | Format kode |
| `cargo clippy` | Lint kode |
| `cargo clean` | Hapus folder target/ |
| `cargo add serde` | Tambah dependency |

### Debug vs Release Build

```bash
# Debug build — compile cepat, runtime lambat, dengan debug info
cargo build
# Output: target/debug/nama_project

# Release build — compile lambat, runtime cepat, optimized
cargo build --release
# Output: target/release/nama_project
```

| Aspek | Debug | Release |
|-------|-------|---------|
| Compile time | Cepat | Lambat |
| Runtime performance | Lambat | Cepat |
| Binary size | Besar | Kecil |
| Debug info | Ada | Tidak ada |
| Optimizations | Off | On |

## Macro println!

`println!` adalah macro untuk output ke stdout:

```rust
fn main() {
    // Basic string
    println!("Hello, world!");

    // Format dengan placeholder {}
    let name = "Rust";
    println!("Hello, {}!", name);

    // Inline format (Rust 1.58+)
    println!("Hello, {name}!");

    // Multiple values
    let x = 10;
    let y = 20;
    println!("{} + {} = {}", x, y, x + y);
    println!("{x} + {y} = {}", x + y);

    // Debug print dengan {:?}
    let arr = [1, 2, 3];
    println!("Array: {:?}", arr);

    // Pretty print dengan {:#?}
    println!("Array:\n{:#?}", arr);

    // Padding dan alignment
    println!("{:>10}", "right");   // "     right"
    println!("{:<10}", "left");    // "left      "
    println!("{:^10}", "center");  // "  center  "

    // Number formatting
    println!("{:05}", 42);         // "00042"
    println!("{:.2}", 3.14159);    // "3.14"
    println!("{:b}", 42);          // "101010" (binary)
    println!("{:x}", 255);         // "ff" (hex)
    println!("{:o}", 8);           // "10" (octal)
}
```

### Variasi Print Macro

| Macro | Output |
|-------|--------|
| `print!()` | Print tanpa newline |
| `println!()` | Print dengan newline |
| `eprint!()` | Print ke stderr tanpa newline |
| `eprintln!()` | Print ke stderr dengan newline |
| `format!()` | Return String (tidak print) |

```rust
fn main() {
    print!("No newline");
    println!(" — with newline");

    eprintln!("This goes to stderr");

    let s = format!("Hello, {}!", "Rust");
    println!("{s}"); // "Hello, Rust!"
}
```

## Comments

```rust
fn main() {
    // Single-line comment

    /* Multi-line
       comment */

    /// Doc comment untuk item di bawahnya (generates documentation)
    /// Supports **Markdown** formatting
    ///
    /// # Examples
    /// ```
    /// let x = add(1, 2);
    /// assert_eq!(x, 3);
    /// ```
    fn add(a: i32, b: i32) -> i32 {
        a + b
    }

    //! Module-level doc comment (biasanya di awal file)

    let x = add(1, 2);
    println!("{x}");
}
```

## Menambah Dependencies

```bash
# Tambah dependency menggunakan cargo add
cargo add serde --features derive
cargo add tokio --features full

# Atau edit Cargo.toml manual:
# [dependencies]
# serde = { version = "1.0", features = ["derive"] }
# tokio = { version = "1", features = ["full"] }
```

Setelah menambah dependency, `cargo build` akan otomatis download dan compile.

## Bank Soal

### Soal 1 — Hello Name (Mudah)

Buat program yang mencetak nama kamu dengan format:

```
Nama saya: [NAMA]
Saya sedang belajar Rust!
```

**Jawaban:**

```rust
fn main() {
    let nama = "Budi";
    println!("Nama saya: {nama}");
    println!("Saya sedang belajar Rust!");
}
```

### Soal 2 — Format Table (Mudah)

Cetak tabel sederhana menggunakan `println!` dengan padding:

```
| Bahasa     | Tahun |
|------------|-------|
| Rust       |  2010 |
| Go         |  2009 |
| Python     |  1991 |
```

**Jawaban:**

```rust
fn main() {
    println!("| {:<10} | {:>5} |", "Bahasa", "Tahun");
    println!("|{:-<12}|{:-<7}|", "", "");
    println!("| {:<10} | {:>5} |", "Rust", 2010);
    println!("| {:<10} | {:>5} |", "Go", 2009);
    println!("| {:<10} | {:>5} |", "Python", 1991);
}
```

### Soal 3 — Debug Print (Mudah)

Buat program yang mendeklarasikan beberapa variabel dan print semuanya menggunakan debug format `{:?}`:

**Jawaban:**

```rust
fn main() {
    let angka = 42;
    let pi = 3.14159;
    let aktif = true;
    let tuple = (1, "hello", 3.14);
    let array = [10, 20, 30, 40, 50];

    println!("angka: {:?}", angka);
    println!("pi: {:?}", pi);
    println!("aktif: {:?}", aktif);
    println!("tuple: {:?}", tuple);
    println!("array: {:#?}", array);
}
```

### Soal 4 — Cargo Project (Sedang)

Buat project baru bernama `kalkulator`, lalu:
1. Tambahkan dependency `colored = "2"`
2. Buat program yang mencetak "Kalkulator Rust" dengan warna

**Jawaban:**

```bash
cargo new kalkulator
cd kalkulator
cargo add colored
```

```rust
use colored::*;

fn main() {
    println!("{}", "Kalkulator Rust".green().bold());
    println!("{}", "================".white());
    println!("2 + 3 = {}", format!("{}", 2 + 3).cyan());
}
```

### Soal 5 — Number Base Conversion (Sedang)

Buat program yang menerima angka 255 dan mencetaknya dalam format desimal, biner, oktal, dan heksadesimal.

**Jawaban:**

```rust
fn main() {
    let num = 255;
    println!("Desimal:     {}", num);
    println!("Biner:       {:b}", num);
    println!("Biner (pad): {:08b}", num);
    println!("Oktal:       {:o}", num);
    println!("Hex (lower): {:x}", num);
    println!("Hex (upper): {:X}", num);
    println!("Hex (pad):   {:#06x}", num);
}
```
