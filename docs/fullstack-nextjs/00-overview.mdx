# Fullstack Next.js Tutorial

> Panduan lengkap membangun aplikasi fullstack dengan Next.js 15, TailwindCSS, Shadcn UI, Drizzle ORM, dan Better Auth

---

## 📋 Project Overview

**Fullstack Next.js** adalah tutorial komprehensif untuk membangun aplikasi web modern dengan stack teknologi terupdate. Tutorial ini mencakup setup project, database dengan Drizzle ORM, authentication dengan Better Auth, UI components dengan Shadcn UI, dan state management dengan Zustand.

| Aspek | Detail |
|-------|--------|
| **Nama Proyek** | Fullstack Next.js Tutorial |
| **Arsitektur** | Next.js 15 App Router + Server Actions + Client Components |
| **Target** | Developer yang ingin mempelajari fullstack development dengan Next.js |
| **Pendekatan** | Step-by-step dari setup hingga deployment |

## 🚀 Mengapa Stack Ini?

### Next.js 15
- App Router untuk server/client component separation
- Server Actions untuk mutations tanpa API routes
- Turbopack untuk development speed

### Drizzle ORM
- Type-safe dari awal hingga akhir
- Lightweight - hanya ~30KB
- Migration yang predictable

### Better Auth
- Type-safe
- Plugin ecosystem yang rich
- Framework agnostic

### Shadcn UI
- Copy-paste ke project (bukan npm package)
- Penuh customization
- A11y yang baik

### Zustand
- Simple API (hanya hooks)
- No boilerplate
- Transient updates tanpa re-render

## 📋 Quick Reference

### Project Structure

```
my-app/
├── src/
│   ├── app/                 # App Router pages
│   │   ├── api/            # API routes
│   │   ├── (auth)/         # Auth pages
│   │   ├── dashboard/      # Protected pages
│   │   ├── layout.tsx
│   │   └── page.tsx
│   ├── components/         # React components
│   │   ├── ui/            # Shadcn components
│   │   └── ...
│   ├── lib/               # Utilities
│   │   ├── db.ts          # Drizzle client
│   │   ├── auth.ts        # Better Auth config
│   │   └── ...
│   ├── stores/            # Zustand stores
│   └── actions/           # Server Actions
├── drizzle/               # Migrations
├── public/               # Static assets
├── drizzle.config.ts
├── next.config.ts
└── tailwind.config.ts
```

### Environment Variables

```bash
# .env.local
DATABASE_URL=postgresql://user:password@localhost:5432/mydb
BETTER_AUTH_SECRET=your-secret-key
BETTER_AUTH_URL=http://localhost:3000
```

### Common Commands

```bash
# Setup
bun create next-app@latest my-app --yes
cd my-app

# Database
bunx drizzle-kit push
bunx drizzle-kit generate

# Development
bun dev
bun build
```
