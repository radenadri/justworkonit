# Setup Project Next.js 15

> Inisialisasi project Next.js 15 dengan Bun, TailwindCSS, dan TypeScript

---

## 📋 Langkah-langkah Setup

### 1. Install Next.js dengan Bun

Gunakan `create-next-app` untuk inisialisasi project:

```bash
# Buat project baru
bun create next-app@latest my-app --yes

# Atau dengan opsi interaktif
bun create next-app@latest my-app
```

Pilih opsi berikut:
- TypeScript: **Yes**
- ESLint: **Yes**
- TailwindCSS: **Yes**
- `src/` directory: **Yes**
- App Router: **Yes**
- Import alias: **@/***

### 2. Install Dependencies Tambahan

```bash
cd my-app

# Install dependencies untuk project
bun add lucide-react clsx tailwind-merge
```

## ⚙️ Konfigurasi TailwindCSS

### tailwind.config.ts

File ini sudah di-generate oleh Next.js, tapi pastikan menggunakan setup terbaru:

```typescript
import type { Config } from "tailwindcss";

const config: Config = {
  content: [
    "./src/pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./src/components/**/*.{js,ts,jsx,tsx,mdx}",
    "./src/app/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {
      colors: {
        background: "hsl(var(--background))",
        foreground: "hsl(var(--foreground))",
        card: {
          DEFAULT: "hsl(var(--card))",
          foreground: "hsl(var(--card-foreground))",
        },
        primary: {
          DEFAULT: "hsl(var(--primary))",
          foreground: "hsl(var(--primary-foreground))",
        },
        secondary: {
          DEFAULT: "hsl(var(--secondary))",
          foreground: "hsl(var(--secondary-foreground))",
        },
        muted: {
          DEFAULT: "hsl(var(--muted))",
          foreground: "hsl(var(--muted-foreground))",
        },
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
        destructive: {
          DEFAULT: "hsl(var(--destructive))",
          foreground: "hsl(var(--destructive-foreground))",
        },
        border: "hsl(var(--border))",
        input: "hsl(var(--input))",
        ring: "hsl(var(--ring))",
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
        sm: "calc(var(--radius) - 4px)",
      },
    },
  },
  plugins: [],
};

export default config;
```

## 🎨 Setup CSS Variables

### src/app/globals.css

Setup CSS variables untuk theming:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  :root {
    --background: 0 0% 100%;
    --foreground: 222.2 84% 4.9%;
    
    --card: 0 0% 100%;
    --card-foreground: 222.2 84% 4.9%;
```

## 🛠️ Utility Functions

### src/lib/utils.ts

Buat utility untuk className merging:

```typescript
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```
