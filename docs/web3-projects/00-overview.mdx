# Web3 Projects Implementation Guide

> Panduan lengkap membangun aplikasi blockchain dari beginner hingga expert

---

## 📋 Project Overview

**Web3 Projects** adalah kumpulan 4 project pembelajaran yang dirancang untuk membantu developer memahami pengembangan aplikasi blockchain secara bertahap. Dimulai dari konsep dasar hingga implementasi sistem governance yang kompleks.

| Aspek | Detail |
|-------|--------|
| **Nama Proyek** | Web3 Learning Path |
| **Arsitektur** | Smart Contracts (Solidity) + Frontend (Next.js + wagmi/viem) |
| **Target** | Developer yang ingin mempelajari Web3 development |
| **Pendekatan** | Tingkat kesulitan: Beginner → Intermediate → Advanced → Expert |

---

## ✅ Prerequisites

Sebelum memulai, pastikan Anda memiliki:

### Tools
- [ ] Node.js 18+ terinstall
- [ ] Git terinstall
- [ ] Metamask extension di browser

### ⚠️ Catatan Penting

**wagmi v3 + RainbowKit Compatibility:**
- Per Januari 2026, RainbowKit **belum sepenuhnya support wagmi v3**
- Masih ada issue yang belum fix: [GitHub Issue #2626](https://github.com/rainbow-me/rainbowkit/issues/2626)
- **Rekomendasi:** Tetap gunakan wagmi v2 untuk saat ini dengan RainbowKit v2.2.x
- Alternatif lain: Gunakan `wagmi/connectors` langsung tanpa RainbowKit jika butuh wagmi v3

### Akun
- [ ] Akun Alchemy/Infura (untuk testnet RPC)
- [ ] Akun Etherscan (untuk verifikasi contract)
- [ ] Akun Pinata (untuk IPFS - optional)

### Pengetahuan Dasar
- [ ] Pemahaman dasar JavaScript/TypeScript
- [ ] Pemahaman dasar React/Next.js
- [ ] Pemahaman dasar tentang blockchain dan Ethereum

## 🚀 Getting Started

### Untuk Pemula

Mulai dari **Project 01 (Todo List)** - ini adalah project paling sederhana yang memperkenalkan:
- Write operations ke smart contract
- Read operations dari smart contract
- Event listening untuk real-time updates

```
Beginner → Intermediate → Advanced → Expert
   (01)   →     (02)     →   (03)   →   (04)
```

Setiap project membangun pengetahuan dari project sebelumnya.
